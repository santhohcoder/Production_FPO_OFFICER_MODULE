Declared "Item Category" of this Article ID is "GIFTS"

1. System by default assesses GIFTS under CTH 98049000 at Tariff Rate.
2. If found otherwise, AO may change the CTH / Notification from the dropdown as found appropriate.


Declared "Item Category" of this Article ID is "DOCUMENTS"

1. System by default assesses Documents under CTH 98049000 as goods imported for personal use.
2. Please note that "Printed Books" are not covered under CTH 9804.
3. If found otherwise, AO may change the CTH / Notification from the dropdown as found appropriate.


Declared "Item Category" of this Article ID is "OTHERS"

1. System by default assesses Other Articles under CTH 98049000 as goods imported for personal use.
2. If found otherwise, AO may change the CTH / Notification from the dropdown as found appropriate.


Declared "Item Category" of this Article ID is "SALE OF GOODS"

1. System by default assesses Sale of Goods Articles under CTH 98049000 as goods imported for personal use.
2. Please note that "Printed Books" are not covered under CTH 9804.
3. If sale is B2B, AO may raise query to move to Commercial Imports Queue to process further.
4. If goods are claimed as Life Saving Drugs, applicable CTH 98041000 process appropriately.
5. If found otherwise, AO may change the CTH/ Notification from the dropdown as found appropriate


Declared "Item Category" of this Article ID is "COMMERCIAL SAMPLE"

1. System by default assesses Commercial Sample under CTH 98049000 as goods imported for personal use.
2. If import is B2B, AO may raise query to move to Commercial Imports Queue to process further.
3. If found otherwise, AO may change the CTH / Notification from the dropdown as found appropriate.


Declared "Item Category" of this Article ID is "RETURNED GOODS"

1. System by default assesses Returned Goods under CTH 98049000 as goods imported for personal use.
2. If the goods are postal articles returned for non-delivery etc, AO may assess accordingly.
3. If found otherwise, AO may change the CTH / Notification from the dropdown as found appropriate.
